# SpriteSlice v2.4 · 精灵图处理工具

直接使用：https://arminosi.github.io/SpriteSlice/

PNG 序列转 GIF 工具（可选）：https://gif.qwq.team/

## 📋 项目简介

- 一个纯前端的精灵图处理工具：切割精灵图、拖拽调整顺序、导出切片；也支持 GIF 拆帧合成精灵图并导出。
- 所有数据都在本地处理：图片不会上传到服务器；历史文件保存在浏览器本地存储中。

## ✨ 主要功能

### 🧩 精灵图切片（PNG/JPEG/WebP）
- **分割参数**：行数/列数（1-50）、起始序号、字号
- **排序方式**：正常 / 倒序 / 奇偶行蛇形（与导出顺序一致）
- **预览模式**：
  - 图像预览：可选择“仅预览显示序号”（不影响导出）
  - 可拖拽网格：拖拽重排顺序、右键删除图块、支持一键恢复
- **导出**：一键导出切片 `ZIP`（文件名类似 `SpriteSlice_export_YYYY_MM_DD_HH_MM_SS.zip`）

### 🎞️ GIF 合成精灵图（GIF）
- **自动处理**：解析 GIF 帧并合成精灵图（spritesheet）
- **导出**：导出合成后的精灵图 `PNG`
- 说明：GIF 模式下不会导出 ZIP 切片（因为本质是“合成精灵图”而不是“切割切片”）

### 📁 历史记录
- **自动保存**：导出后会记录到“历史文件”
- **管理能力**：下载 / 预览 / 删除 / 清空
- 历史预览会从 ZIP 中读取 `tiles/` 目录下的图片进行展示

### 🌍 国际化与快捷键
- **中英文切换**：页面文案与提示同步切换
- **快捷键**：
  - `Ctrl+O`：选择图片
  - `Ctrl+P`：生成预览
  - `Ctrl+S`：导出（按钮会随当前图片类型显示“导出切片/导出精灵图”）
  - `ESC`：关闭历史窗格 / 关闭预览弹层

## 🚀 快速开始

### 环境要求
- 现代 Web 浏览器（Chrome / Firefox / Safari / Edge）

### 本地运行（推荐）

在项目根目录启动任意静态服务器即可：

```bash
# Python
python3 -m http.server 8000

# 或 Node
npx http-server
```

打开：`http://localhost:8000`

### 直接打开
也可以直接双击 `index.html` 用浏览器打开（部分浏览器对本地文件有跨域/权限限制，推荐还是用本地服务器）。

## 📖 使用指南

### 流程 A：切片导出（PNG/JPEG/WebP）
1. 选择图片（精灵图）
2. 设置行数/列数/排序方式等参数
3. 点击“生成预览”确认效果（可切换预览模式并拖拽调整顺序）
4. 点击“导出切片（ZIP）”下载

### 流程 B：GIF 导出（GIF）
1. 选择 GIF
2. 工具会自动解析并合成精灵图预览
3. 点击“导出精灵图（PNG）”下载

## 🧰 依赖说明

- `JSZip`：用于生成 ZIP（切片导出 + 历史预览解压）。如果 CDN 被拦截或加载失败：
  - 切片导出/历史预览会提示缺少依赖
  - 但页面仍可正常打开与预览（GIF 合成精灵图导出不依赖 JSZip）
